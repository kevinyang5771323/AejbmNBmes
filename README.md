# 宠物医院管理系统 java823

## 项目概述

宠物医院管理系统是一款集用户、医生、管理员三种角色于一体的多模块在线服务平台。本项目采用前后端分离的技术架构，前端使用Vue和uniapp，后端使用Springboot，确保系统的高效稳定运行。

## 技术栈

- **开发环境**：IDEA、JDK1.8、Maven、Node.js
- **技术框架**：Java、SpringBoot、MyBatis-Plus、Vue、uniapp
- **数据库**：MySQL
